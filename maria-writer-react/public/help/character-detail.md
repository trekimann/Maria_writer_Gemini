# Character Details

The **Character Details** page provides a comprehensive view of a specific character in your story.

## Overview
This view consolidates all information about a character, including their profile, description, related events, and wherever they are mentioned in your chapters.

## Features

### Profile Header
- **Avatar**: Displays the character's image or initial.
- **Basic Info**: Shows name, age (calculated based on current date), gender, and dates of birth/death.
- **Tags**: Any tags associated with the character.

### Tabs
The view is divided into two tabs:

1. **Info & Events**
   - **Description**: The full character description.
   - **Aliases / Nicknames**: Alternate names for the character.
   - **Related Events**: A timeline of events where this character is involved. Click an event to view its details.

2. **Mentions**
   - Lists all chapters where this character is mentioned.
   - Shows a snippet of the text surrounding the mention.
   - Click a mention to jump directly to that chapter in the editor.

## Actions
- **Back**: Returns to the main character list.
- **Edit**: Opens the character edit modal to update details like name, description, tags, etc.
- **Edit Picture**: (In Edit mode) You can change the character's avatar URL.
