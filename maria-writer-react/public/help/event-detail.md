# Event Details

The **Event Details** page allows you to examine a specific event in depth.

## Content
- **Hero Image**: If an image is associated with the event, it appears at the top.
- **Title**: The name of the event.
- **Metadata**: 
  - **Date**: The chronological point in the story.
  - **Character Count**: Number of characters tagged in this event.
- **Description**: The full text description of what occurs.
- **Involved Characters**: A list of avatars and names for all characters present in this event.

## Actions
- **Back**: Returns to the main Event List.
- **Edit**: Opens the event editor to modify details, change the date, or add/remove characters.
- **Character Navigation**: Clicking on any character in the "Involved Characters" list will navigate you to that character's detail page.
