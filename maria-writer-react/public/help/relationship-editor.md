# Relationship Editor

The Relationship Editor allows you to define or modify connections between characters.

## Fields

- **Relationship Type**: Select the nature of the relationship (e.g., Family, Friend, Rival).
- **Characters**: Select the characters involved in this relationship. You must select at least **two** characters.
- **Description**: Add details about the relationship, such as how they met or the current state of their dynamic.
- **Start Date**: When the relationship began.
- **End Date**: When the relationship ended (if applicable).

## Actions

- **Create/Update**: Saves the relationship.
- **Cancel**: Closes the dialog without saving.
