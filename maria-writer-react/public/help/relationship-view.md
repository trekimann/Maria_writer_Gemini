# Relationship View

The Relationship View allows you to visualize and manage the connections between characters within your story.

## Visualization Modes

Use the toggle buttons in the top right to switch between views:
- **Graph View**: An interactive network graph showing characters as nodes and relationships as links.
  - **Drag** nodes to rearrange them.
  - **Click** a node or line to select it.
  - **Scroll** to zoom in/out.
- **Split View**: Shows the graph and list side-by-side.
- **List View**: Displays a detailed list of all relationships.

## Managing Relationships

- **Add Relationship**: Click the **+ Add Relationship** button to define a new connection.
- **Edit**: Click on a relationship in the list or graph to modify it.
- **Delete**: Use the delete button on relationship cards in the list view.

## Graph Controls
- **Fit View**: Centers the graph on the screen.
- **Toggle Physics**: Enables or disables the force-directed layout simulation.
- **Legend**: Shows the color coding for different relationship types.
