# Event Editor

The **Event Editor** allows you to create or modify events in your story's timeline.

## Fields

### Visuals
- **Image**: Click the placeholder to upload an image representing this event.

### Basic Info
- **Event Title**: The name of the event (Required).
- **Date**: The chronological date of the event. Format: `dd/MM/yyyy HH:mm:ss`. Events without dates will likely appear at the end or in a separate list depending on view.
- **Description**: Detailed text about what happens in this event.

### Characters
- **Characters Involved**: Check the boxes for all characters who participate in this event. This links the event to their individual timelines.

## Actions
- **Save Event**: Persists your changes.
- **Cancel**: Closes the modal without saving.
- **Delete**: (Edit Mode Only) Permanently removes the event.
