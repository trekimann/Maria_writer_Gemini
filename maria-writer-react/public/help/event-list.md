# Event List

The **Event List** provides a chronological or list-based view of all significant plot points and occurrences in your story.

## Features

### Management
- **Search**: Use the search bar to filter events by title or description content.
- **Add Event**: Click the **+ Add Event** button to create a new timeline entry.

### Event Cards
Each event in the list displays:
- **Title**: The name of the event.
- **Date**: When the event occurs (if set).
- **Description**: A brief summary of what happens.
- **Characters**: Tags showing which characters are involved.

## Interaction
- **View Details**: Click anywhere on an event card to open the detailed view.
