# Chapter Metadata

Manage the metadata for the current chapter.

## Fields
- **Title**: The title of the chapter. Updating this will also update the H1 heading in the chapter content.
- **Setting Date**: The date and time when the chapter takes place. Format: `dd/MM/yyyy HH:mm:ss`.
- **Related Events**: Select events from the timeline that are related to this chapter.
- **Characters**: Select characters that appear or are mentioned in this chapter.

## Tips
- Use the date field to order your chapters chronologically in the timeline view.
- Linking events helps in tracking the story arc.
