# Character Editor

The **Character Editor** allows you to create new characters or edit existing ones.

## Fields

### Basic Information
- **Name**: The display name of the character (Required).
- **Picture**: Click the avatar placeholder to upload an image, or "Remove" to clear it.
- **Age**: Text field for character's age description used in display.
- **Gender**: Text field for gender.

### Key Dates
- **DOB**: Date of Birth. Format: `dd/MM/yyyy HH:mm:ss`. Used for timeline calculations.
- **Died**: Date of Death. Format: `dd/MM/yyyy HH:mm:ss`. Marks character as deceased.

### Classification
- **Tags**: Keywords to categorize the character (e.g., "Protagonist", "Villain"). Press Enter to add a tag.
- **Nicknames/Aliases**: Alternative names the character is known by.
- **Highlight Color**: Choose a color for the character's avatar border and tag highlights.

### Biography
- **Description/Bio**: Free-text field for character background, personality, and notes.

### Relationships
- **Parents**: (New Character Only) Select existing characters to automatically create parent-child relationships.

## Actions
- **Save Character**: Persists your changes.
- **Cancel**: Closes the modal without saving.
- **Delete**: (Edit Mode Only) Permanently removes the character and their relationships.
